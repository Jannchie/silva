# 完成度（finish）评分细则

**版本**: `finish-v1`（标注事件的 rubric_version 引用此值；修订时 bump 至 finish-v2 并保留旧文）

## 维度定义

sketch→精修的精修程度、细节装饰的精致度。

**不属于本维度**：纯格式问题（3koma、漫画分格——留给 danbooru tags）、配色品味、构图。

## 切点（二元 scale=2）

- **2（好）**: 精修完整，细节装饰让我想放大看。
- **1（不好）**: 草稿感/速涂感/完整但平。

## 三元切点（scale=3，形态实验用）

- **3（好）**: 精修完整，细节让我想放大看。
- **2（中）**: 完整但平，无装饰亮点。
- **1（差）**: 草稿感/速涂感/未完成。
