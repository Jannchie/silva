# 构图（composition）评分细则

**版本**: `composition-v1`（标注事件的 rubric_version 引用此值；修订时 bump 至 composition-v2 并保留旧文）

## 维度定义

广义演出：姿势动态、镜头角度、场景安排。不细分子轴——判断时综合，
但 checklist 按子轴列（姿势是否生硬？角度有没有想法？场景是否服务主体？）。

**不属于本维度**：画质、上色质量、完成度。

## 切点（二元 scale=2）

- **2（好）**: 演出有想法（动态/角度/布景任一出彩）。
- **1（不好）**: 站桩/呆板/常规证件照式构图。

## 三元切点（scale=3，形态实验用）

- **3（好）**: 演出有想法（动态/角度/布景任一出彩）。
- **2（中）**: 常规但不呆板。
- **1（差）**: 站桩/呆板/证件照式。
